# プライバシーポリシー

**制定日:** 2026年1月2日

## 1. アプリ概要
本アプリ「デコ雀トレ」（以下、当アプリ）は、麻雀の点数計算や多面待ちのトレーニングを行うためのアプリです。

## 2. 利用者情報の取得
当アプリでは、より良いサービス提供および広告配信のために、以下の第三者サービスを利用して利用者情報を取得することがあります。

### AdMob (Google)
当アプリでは、広告配信のためにGoogle AdMobを使用しています。
AdMobは、ユーザーの興味に応じた広告を表示するために、Cookie、デバイス識別子（IDFAなど）、位置情報データ（おおよその位置）を使用することがあります。

Googleのプライバシーポリシーおよびデータの取り扱いについては、以下のリンクをご参照ください。
*   [Google プライバシーポリシー](https://policies.google.com/privacy)
*   [Google のサービスを使用するサイトやアプリから収集した情報の Google による使用](https://policies.google.com/technologies/partner-sites)

## 3. 個人情報の利用目的
取得した情報は、以下の目的で利用されます。
*   アプリの機能提供および改善
*   広告の配信および最適化
*   利用状況の分析

## 4. 免責事項
当アプリの利用により生じた損害について、開発者は一切の責任を負いません。
麻雀の役や点数計算のルールについては一般的なものを採用していますが、ローカルルール等により異なる場合があります。
